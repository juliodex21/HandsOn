Tecnologia da Informação

Conheça o MySQL

O MySQL é o banco de dados de cógigo aberto mais popular do mundo. Apesar de seus recursos poderosos, ele é simples de configurar e fácil de usar. Neste curso você irá aprender técnicas de como modelar e projetar banco de dados relacionais usando a tecnologia do MySQL. Além disso, aprederemos juntos como consultar e manipular dados nas tabelas do MySQL usando a Linguagem de Consulta Estruturada (SQL).
